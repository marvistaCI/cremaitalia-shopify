Crema Italia Brand & Document Standards

Version 2.1 · July 2026 · Owner: Crema Italia, LLC · Lutz, Florida, USA · cremaitalia.com · Applies to: all Crema Italia documents (DOCX, PDF, Markdown), web pages, decks, email templates, and partner-facing collateral.

This is the canonical reference. When you ask a tool to build a Crema Italia document, slide, or web page, point it at this file and the accompanying Crema Italia Brand CSS.css and the finalized logo files in Logo Assets/. The look established by the Roaster Onboarding & Partnership Guide is the visual reference; this document codifies it.

What changed in v2.1 (2026-07-14)

One voice rule added (§9): no em-dashes in customer-facing copy, with the semicolon → spaced-hyphen and trailing → ellipsis substitutions. Color, type, logo, and layout grid are unchanged from v2.0; this is a copy-only minor revision.

What changed in v2.0 (artist rebrand, 2026-07-01)

A graphic artist delivered the finalized Crema Italia™ mark (vector master + full lockup set) for the trademark filing. Two brand systems were revised as a result: the palette — Espresso Brown is now #55331B and Crema Gold is #B88348 (replacing the former #3B1F12 / #C46A1F); and the type — the wordmark is set in Montecatini Pro (in the logo art only), and live web/document headings use Marcellus, ending the earlier Cormorant→Lora display-font history. Green, red, cream, and Inter body type are unchanged.

1 · Brand essence

Crema Italia is a small, selective importer of artisan Italian coffee. The visual language must read as:

  • Editorial — long-form, considered, like a fine-press book. Not e-commerce loud.
  • Italian — a quiet nod to the tricolore (green / white / red) without ever shouting it.
  • Warm — cream and coffee tones, never stark white, never grey corporate.
  • Bilingual-ready — every layout must comfortably hold an English column and an Italian column side by side.

When in doubt: fewer elements, more whitespace, smaller logo, larger margins.

2 · Logo

Files: the finalized artist mark lives in Logo Assets/ — a vector master (Art Files/*.ai) plus matched EPS/, PDF/, SVG/, PNG/ exports and generated Web/ derivatives. Three lockups — main (vertical), horizontal, and favicon (cup only) — each in a light-background and a dark-background version.

  • Trademark: the mark is Crema Italia™. The ™ sits as a cap-height superscript after the final “A” in “ITALIA.” Never use ® — Crema Italia is not federally registered yet.
  • Clear space: at least the height of the wordmark “C” on every side. Never crop the tricolore arcs.
  • Minimum size: print 25 mm wide; screen 120 px wide. A favicon-only crop of the cup is acceptable below 120 px.
  • On dark backgrounds: use the dark-background lockup (cream wordmark), e.g. PNG/CI Logo for Dark Background - Transparent.png — never the light mark recolored. Never re-color the cup.
  • Don’t: stretch, rotate, or add drop-shadows; place on photographs without a solid backing; re-typeset the wordmark. Master edits happen in the .ai; re-export the others.

Rule: Crema Gold’s contrast on cream (3.1 : 1) is lower than the terracotta it replaced — it clears WCAG only for large text. Use gold for large headings, short bold accent runs, links, and key numerals; never body copy or small text. When a gold element could fall below ~18 px, set it in Espresso Brown instead.

3.5 Don’t introduce new colors

If a chart or web feature seems to need a new hue, derive it from Crema Light, Espresso, or a desaturated Tricolore Green. The brand should never sprout a sixth or seventh accent.

4 · Typography

Two live families plus the logo face. Hold this line — the document brand depends on it.

4.1 The pairing

RoleFamilyWhere to use
Logo wordmarkMontecatini Pro (Normale Semi-Bold)Logo artwork only — already outlined in the files. Commercial font; licensing unconfirmed for any use beyond the outlined mark.
Display & HeadingsMarcellus (serif, weight 400)Cover title, H1, section headings, web/document display. The free Google-Font stand-in for Montecatini.
Body & UIInter (sans, 400 / 500 / 600)All body text, tables, captions, web UI, buttons, tagline.

Why this pairing: Marcellus echoes the classical Roman capitals of the wordmark and reads as considered and Italian without theatrics; Inter is a calm, neutral workhorse that handles bilingual copy and small sizes without strain. Both are free, web-safe, and embed cleanly in PDF.

Marcellus is roman-only — it ships a single 400 weight with no italic. Two consequences: (1) heading hierarchy is set by size, not weight; use bold Inter for emphasis within text. (2) The English-vs-Italian cue is now carried by the ENGLISH / ITALIANO eyebrow label and column position — not by italic display type. Reserve italics (Inter Italic) for references and short foreign words.

4.2 Sub-heading exception

H2 sub-headings are set in Inter SemiBold (600) in Crema Gold, not in serif. Keep this rule — it visually separates major editorial titles (Marcellus, brown) from in-page section markers (Inter, gold).

4.5 Body type rules

  • Measure: 65–75 characters per line. In a two-column bilingual Letter layout this lands at ~3.1 in / 7.9 cm column width.
  • Paragraph spacing: 6 pt before, no first-line indent. Never both indent and space.
  • Hyphenation: off for English and Italian; a slightly ragged right reads better, and Italian breaks awkwardly.
  • Numerals: lining, tabular in tables and prices; proportional elsewhere.
  • Italics carry meaning — Italian-language references and short foreign words (Inter Italic). Don’t italicize for emphasis; use bold.

4.6 Font fallbacks

Display: "Marcellus", "Georgia", serif · Body: "Inter", "Helvetica Neue", "Arial", sans-serif. For Word documents, ship Marcellus and Inter as embedded named styles; if a recipient lacks them, Word falls back to Georgia and Calibri — close enough to preserve hierarchy.

5.2 Page chrome

  • Top: two short rules — green #0E7A3A left, red #C8342B right, ~1.6 in long, 1.2 pt thick. Wordmark “Crema Italia” sits left in 9–11 pt Marcellus; running title sits right in 9 pt Inter.
  • Bottom: single hairline rule #D9D2C2, 0.5 pt. Below it, footer “Crema Italia, LLC · Lutz, Florida, USA · cremaitalia.com” left, “Page N” right, both 8.5 pt Inter Mute.
  • Cover: logo top-center, centered title block, three-column meta strip (Issued by / Audience / Status) above a hairline, and a 6 mm green-cream-red flag strip at the foot.

5.3 Bilingual columns

  • Always English left, Italian right.
  • Column eyebrow labels (ENGLISH, ITALIANO) in Crema Gold, set above the H1.
  • A 0.5 pt hairline runs the full text height between columns.
  • Because Marcellus has no italic, the English/Italian distinction is carried by the eyebrow label and column position — not by italic display type (a change from v1.0).

5.4 Lists, tables, callouts

  • Bulleted lists: • (U+2022), 0.18 in indent, 0.12 in hanging indent. No nested bullets in body content; if you need a second level, restructure into a table.
  • Tables: Espresso-brown header bar #55331B with cream text; body rows on Ivory #FFFFFF; row separators #D9D2C2. No vertical lines.
  • Callouts: cream box #FBF8F1 on ivory page, 1 pt left rule in Crema Gold #B88348, 0.25 in inner padding.
  • References: italic caption style, prefixed “Reference:” / “Riferimento:”.

6 · Web styling

The accompanying Crema Italia Brand CSS.css (v1.2) is the canonical implementation. It exposes every value above as a CSS custom property and includes ready-made classes for .eyebrow, .bilingual, .callout, .flag-rule, and .btn-primary.

6.1 Web rules

  • Max content width 72 ch for long-form, 1120 px for landing pages.
  • Load fonts: Marcellus + Inter from Google Fonts.
  • Buttons: Crema Gold fill, Cream text, 8 px radius, 12 px / 24 px padding, no shadow.
  • Hover: darken Crema by ~8% — --ci-crema-hover: #9C6E3C.
  • Focus ring: 2 px Crema Gold with 2 px Cream offset — required for accessibility.
  • Never animate gratuitously. A 200 ms color transition on links is the ceiling.

7 · Data visualization

  • Single-series: Crema Gold #B88348.
  • Two-series: Crema Gold #B88348 and Espresso Soft #6B4A38.
  • Three-to-five series: Crema Gold, Espresso Soft, Tricolore Green #0E7A3A, Espresso Brown #55331B, Crema Light #E8A86A — in that order.
  • Background: Cream. Gridlines: Hairline #D9D2C2, dashed, 0.5 pt. Axis labels: Inter 9 pt, Mute.
  • Title above the chart in Inter SemiBold 11 pt Espresso — state the insight, not the variable.
  • No 3D, no pie charts with more than 3 wedges, no dual axes.

8 · Imagery & photography

  • Style: natural light, low saturation, narrow depth of field. Espresso cups, roasting drums, hands at work, Tuscan landscape.
  • Don’t: stock photos of generic baristas, vector illustrations, emoji, decorative coffee-bean borders.
  • Treatment: images sit on Cream with a 1 px Hairline border and 0.25 in margin. No drop-shadows.
  • Aspect ratios: 3:2 landscape product shots, 1:1 roaster portraits, 16:9 hero web banners.

9 · Voice & tone (brief)

  • Plain, direct, deferential. “We choose a small number of roasters whose work represents the craft of Italian roasting at its best.” Not: “We hand-pick the world’s most exclusive coffee artisans.”
  • Bilingual parity. Every roaster-facing artifact ships in English and Italian. Italian is the controlling version when handed to an Italian roaster.
  • Honour formal address in Italian. Use Voi / Vi / Vostro, capitalized.
  • Numbers and units. Always include both metric (250 g, 1 kg) and U.S. customary (8.82 oz, 2.20 lb) on weight references.
  • No em-dashes in customer-facing copy. Storefront and marketing text — product and page copy, forms, email templates, the coming-soon page — never uses the em-dash, which reads as machine-written. Where a semicolon would serve, use a spaced hyphen ( - ); where a sentence trails into a follow-on thought, use an ellipsis (...); if a case is genuinely ambiguous, ask. Customer-facing copy only — internal documents, this one included, may still use them.

10 · File naming & versioning

  • Documents: Crema_Italia_<Topic>_v<MAJOR>.<MINOR>.{docx|pdf}.
  • Logos: organized by lockup and format in Logo Assets/ — Art Files/ (.ai master), EPS/, PDF/, SVG/, PNG/, Web/. Lockups: main, horizontal, favicon — each light and dark.
  • CSS / templates: Crema Italia Brand CSS.css, kept under version control.
  • Increment minor (2.0 → 2.1) for copy fixes. Increment major (2.x → 3.0) for any change to color, type, or layout grid.
  • Always use the greatest version-number variant of a file (_v2 > _v1).

11 · Quick checklist before sharing any artifact

  • Logo: correct file & lockup, clear space respected, minimum size honored, ™ present (never ®).
  • Background is Cream #FBF8F1, not white (except inside tables/cards).
  • Headings in Marcellus, body in Inter; wordmark artwork only in Montecatini.
  • Crema Gold used only on large headings, links, key numerals — never body or small text.
  • Tricolore green and red appear only as thin rules, never as fills.
  • Bilingual layout: English left, Italian right; English/Italian cue via eyebrow label, not italic.
  • Footer carries “Crema Italia, LLC · Lutz, Florida, USA · cremaitalia.com”; page numbers present.
  • Body type ≥ 10.5 pt (print) / 17 px (web); contrast checked (gold large-only).
  • No introduced colors, no decorative graphics, no stretched logo.
